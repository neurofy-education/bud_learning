# Bud Learning

A CLI to help you learn faster and better with AI help.

## Installation on MacOS

```bash
brew install imagemagick
git clone https://github.com/neurofy-education/bud_learning
cd bud_learning
pip install -e .
cp .env.template .env
# Edit .env with your credentials
```

## Usage

```bash
bud-learning -h
```
